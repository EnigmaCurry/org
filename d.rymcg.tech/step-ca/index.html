Step-CA

Step-CA is a secure, online, self-hosted Certificate Authority (CA). Its purpose is to issue (sign) X.509 (TLS) certificates, and to securely store the private key for the root CA. It can issue certificates manually using the step-cli tool, or Traefik can request them automatically via the ACME protocol.
